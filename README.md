# Guessthesongspotify-DrieKraaien
Reveals per click the artist, title and year of release. One more tap is next song. Optional timer. Like the game hitster but with spotify playlists instead of cards. 100% vibed.

It only works for up to 5 accounts in development mode.

If you want to play locally you need a spotify premium account and python.
Become developer, create app with Web API and Web Playback SDK checked.
Go to user management to add accounts
Back to basic information
Add http://127.0.0.1:8080 to 'Redirect urls'
Copy client id
Download index.html over here to any folder and open it in notepad
Replace the client_id with yours
Replace the url on the next line with http://127.0.0.1:8080
Save and close
Righ click in the folder, Open in terminal
type: python -m http.server 8080
Hit enter
Browse to http://127.0.0.1:8080
Play
